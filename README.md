# 🌾 Farmers Empowerment Platform

A web-based platform designed to support farmers by providing access to agricultural training programs, government schemes, bank loan comparisons, financial guidance, and an online marketplace for better farm management and income growth.

This project aims to bridge the gap between farmers and modern agricultural resources by creating a simple, user-friendly platform where farmers can learn, connect, compare financial options, and access important agricultural services efficiently.

---

## 🚀 Key Features

### 💰 Bank Loan Comparison
- Compare loan options from different banks
- View interest rates and benefits
- Helps farmers choose better financial support options

### 📋 Government Schemes (Yojanas)
- Access different government schemes in one place
- Easy navigation of farmer welfare programs
- Information about subsidies, support plans, and benefits

### 🤝 Financial Support Guidance
- Guidance for agricultural financial planning
- Support for loans, subsidies, and farming investments
- Helps farmers make better financial decisions

### 👨‍🌾 Farmer Registration & Login
- Secure farmer signup and login system
- User authentication with session handling
- Personalized dashboard access

### 📚 Training Programs
- Access to agricultural training modules
- Skill development resources for modern farming techniques
- Program enrollment system

### 🛒 Online Marketplace
- Farmers can explore agricultural products
- Add products to cart
- Checkout functionality
- Order confirmation system

### 📊 Dashboard
- Farmer profile management
- Access enrolled programs
- Track activities and platform usage

### 📧 Email Integration
- Email configuration support for notifications
- Enrollment success and order success confirmations

---

### 🔌 Swagger API Integration
- API documentation using Swagger (OpenAPI)
- Easy testing of backend APIs
- Clear request and response structure
- Better developer experience for future scaling
